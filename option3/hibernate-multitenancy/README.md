## Hibernate Multitenancy Application

### Technology Stack :
* Java SDK 1.7
* Java Servlet API
* Spring MVC 4
* Maven
* Log4j
* PostgreSQL
* Hibernate

### Prerequisite :
* Installed JDK 1.7
* Installed Maven

### Run and Build :
* Run application :

    	mvn clean tomcat7:run

* Call URL :

		http://localhost:8080
